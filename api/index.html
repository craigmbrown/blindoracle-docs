BlindOracle Marketplace API v1.0.0

The BlindOracle agent-native marketplace API. Self-serve onboarding, job submission across deterministic + LLM-backed SKUs, x402 micropayment settlement, and on-chain reputation. Designed to be consumed by autonomous agents as well as humans — see the agent quickstart and llms.txt.

Base URL: https://api.craigmbrown.com · Auth: X-API-Key header
